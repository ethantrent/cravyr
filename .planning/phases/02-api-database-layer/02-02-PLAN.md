Create `apps/api/src/services/geo-cache.ts` with the following implementation:

**Imports:**
```typescript
import NodeCache from 'node-cache';
import ngeohash from 'ngeohash';
import { createClient } from '@supabase/supabase-js';
import { searchNearby, mapPlaceToRestaurant } from './places';
```

**Supabase client:** Create with service role key (`process.env.SUPABASE_SERVICE_ROLE_KEY!`) since this service writes to restaurants and geo_cache tables.

**node-cache instances:**
- `geoCache = new NodeCache({ stdTTL: 3600, checkperiod: 600 })` -- 1 hour hot cache for geo cells
- `budgetCache = new NodeCache({ stdTTL: 86400, checkperiod: 3600 })` -- 24 hour hot cache for budget counter

**Interface:**
```typescript
interface CachedCell {
  restaurants: any[];  // restaurant rows from Supabase
  fetchedAt: string;
}

interface NearbyResult {
  restaurants: any[];
  source: 'memory' | 'database' | 'stale' | 'google_places';
  cache_only?: boolean;
  cache_refreshing?: boolean;
}
```

**Core function `getRestaurantsForLocation(lat: number, lng: number): Promise<NearbyResult>`:**

1. Encode coordinates to geohash: `const hash = ngeohash.encode(lat, lng, 5);`

2. **Layer 1 -- node-cache (hot):**
   ```typescript
   const cached = geoCache.get<CachedCell>(hash);
   if (cached) return { restaurants: cached.restaurants, source: 'memory' };
   ```

3. **Layer 2 -- Supabase geo_cache (cold):**
   ```typescript
   const { data: dbCell } = await supabase
     .from('geo_cache')
     .select('geohash, fetched_at, restaurant_count')
     .eq('geohash', hash)
     .single();
   ```

4. If `dbCell` exists and is fresh (fetched_at < 24 hours ago):
   - Load restaurants from Supabase: `supabase.from('restaurants').select('*').eq('geohash', hash).eq('is_active', true)`
   - Warm node-cache: `geoCache.set(hash, { restaurants, fetchedAt: dbCell.fetched_at })`
   - Return `{ restaurants, source: 'database' }`

5. If `dbCell` exists but is stale (fetched_at >= 24 hours ago) -- **stale-while-refresh per D-03:**
   - Load restaurants from Supabase (same query as step 4)
   - Call `refreshCellInBackground(hash, lat, lng)` -- fire-and-forget (do NOT await)
   - Return `{ restaurants, source: 'stale', cache_refreshing: true }`

6. If no `dbCell` (complete miss):
   - Check budget: if `await isBudgetExhausted()` returns true, return `{ restaurants: [], source: 'memory', cache_only: true }`
   - Otherwise call `fetchAndCacheCell(hash, lat, lng)` synchronously
   - Return `{ restaurants, source: 'google_places' }`

**Helper `fetchAndCacheCell(hash: string, lat: number, lng: number): Promise<any[]>`:**

1. Call `searchNearby(lat, lng)` to get GooglePlace array
2. Increment budget: `await incrementBudgetCount(1)`
3. For each place, call the `upsert_restaurant` RPC function via Supabase:
   ```typescript
   const mapped = mapPlaceToRestaurant(place, hash);
   await supabase.rpc('upsert_restaurant', {
     p_external_id: mapped.external_id,
     p_source: mapped.source,
     p_name: mapped.name,
     p_lng: place.location.longitude,
     p_lat: place.location.latitude,
     p_address: mapped.address,
     p_photo_urls: mapped.photo_urls,
     p_cuisines: mapped.cuisines,
     p_price_level: mapped.price_level,
     p_geohash: hash,
   });
   ```
4. Upsert the geo_cache row:
   ```typescript
   await supabase.from('geo_cache').upsert({
     geohash: hash,
     center_lat: lat,
     center_lng: lng,
     fetched_at: new Date().toISOString(),
     restaurant_count: places.length,
     request_count: 1,
   }, { onConflict: 'geohash' });
   ```
5. Load the full restaurant rows back from Supabase (to get UUIDs and complete data):
   ```typescript
   const { data: restaurants } = await supabase.from('restaurants').select('*').eq('geohash', hash).eq('is_active', true);
   ```
6. Warm node-cache: `geoCache.set(hash, { restaurants: restaurants ?? [], fetchedAt: new Date().toISOString() })`
7. Return `restaurants ?? []`

**Helper `refreshCellInBackground(hash: string, lat: number, lng: number): void`:**
- Mark the cell as refreshing: `supabase.from('geo_cache').update({ is_refreshing: true }).eq('geohash', hash)` (fire-and-forget)
- Call `fetchAndCacheCell(hash, lat, lng)` in a `.then().catch()` chain (not awaited)
- In the `.then()`, update `is_refreshing: false`
- In the `.catch()`, log the error and update `is_refreshing: false`

**Budget functions:**

`getDailyKey(): string` -- returns `budget:${new Date().toISOString().slice(0, 10)}`

`async function isBudgetExhausted(): Promise<boolean>`:
1. Check node-cache first: `const cached = budgetCache.get<number>(getDailyKey())`
2. If cached !== undefined, return `cached >= 500` (per D-12)
3. If not cached, query Supabase: `supabase.from('api_budget').select('request_count').eq('date_key', new Date().toISOString().slice(0, 10)).single()`
4. If row found, warm budgetCache and return `row.request_count >= 500`
5. If no row, return false (new day, no requests yet)

`async function incrementBudgetCount(count: number = 1): Promise<number>`:
1. Get current from node-cache or default 0
2. Increment: `newCount = current + count`
3. Set in node-cache: `budgetCache.set(getDailyKey(), newCount)`
4. Upsert to Supabase api_budget table (fire-and-forget, wrapped in `.catch()` to avoid breaking the flow):
   ```typescript
   supabase.from('api_budget').upsert({
     date_key: new Date().toISOString().slice(0, 10),
     request_count: newCount,
     cache_only_mode: newCount >= 500,
     updated_at: new Date().toISOString(),
   }, { onConflict: 'date_key' }).then(() => {}).catch(console.error);
   ```
5. Return `newCount`

**Staleness check helper:**
```typescript
function isStale(fetchedAt: string, ttlMs: number): boolean {
  return Date.now() - new Date(fetchedAt).getTime() > ttlMs;
}
```

Export: `getRestaurantsForLocation`, `isBudgetExhausted`, `incrementBudgetCount`.
