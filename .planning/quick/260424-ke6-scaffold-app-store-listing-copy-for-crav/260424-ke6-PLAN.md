3. **iOS Description structure** (4000 chars, but target ~1500-2500 — density > length):
   - **First 3 lines (above-the-fold)**: hook + core loop in one breath. Apple truncates at ~170 chars on mobile listing page.
   - **Feature bullets**: 5-7 concrete items, each one verb + noun + payoff.
   - **Privacy one-liner**: "No ads. No cross-app tracking. No data brokers." (echo privacy.html section 7).
